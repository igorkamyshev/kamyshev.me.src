## Content

## Development

```sh
yarn
yarn dev
```

Quality script:

- `yarn s lint` for ESLint + Stylelint
- `yarn s pretty` for Prettier
- `yarn spell` for Yandex.Speller
- `yarn size` for Size Limit

## Release

Netlify uses master-branch for main site.

```sh
yarn
yarn build
```
